# PropertyManager

Administradora open-source de aluguel por temporada (Airbnb). Landing inspirada na estrutura Hostnjoy + calculadora de rendimento estilo Bewave, com identidade própria.

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- lucide-react (ícones)

## Desenvolvimento

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # build de produção
```

## Estrutura da landing

Hero com prova social → logos de parceiros → estimador rápido de renda → calculadora completa → propostas de valor → resultados de mercado → depoimentos → CTA intermediário → serviços → vantagens → cenários reais → FAQ → bairros → outras cidades → CTA final → footer.
